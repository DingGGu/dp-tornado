.. title:: dp-tornado

dp-tornado
==========

MVC Web Application Framework with Tornado, Python 2 and 3

To install the package run::

    $ pip install dp-tornado

Run::

    $ pip install virtualenv
    $ virtualenv ./venv
    $ . ./venv/bin/activate
    $ pip install dp-tornado

    $ dp4p init app_dir
    $ dp4p run app_dir

    or

    $ dp4p init --path app_dir
    $ dp4p run --path app_dir
